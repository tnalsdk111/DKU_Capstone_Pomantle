# 🎮 포맨틀 (Pomentle)

> **"매일 새로운 포즈에 도전하라!"** > AI 기반 실시간 관절 인식 기술을 활용한 일일 포즈 맞추기 게임, **포맨틀**입니다.  
> 사용자의 신체 동작을 정밀하게 분석하여 정답 포즈와의 유사도를 측정하고 기록합니다.

<br>

## 🕹️ Game Play & Rules (게임 방식)

### 1️⃣ 오늘의 정답 포즈 맞추기
- 매일 새로운 **'오늘의 포즈'** 가 출제됩니다.
- 사용자는 정답 포즈가 무엇일지 추측하며 온몸을 움직여 카메라 앞에 섭니다.

### 2️⃣ 실시간 관절 인식 및 AI 유사도 판정
- 카메라로 사진을 촬영하면, AI 엔진이 사용자의 **상체(어깨와 팔꿈치), 왼손, 오른손의 주요 관절 좌표를 실시간으로 추출**합니다.
- 추출된 스켈레톤 각도 데이터를 바탕으로 정답 포즈와의 유사도를 계산합니다.
- 🎯 **정답 조건:** 유사도가 **90%를 넘으면** 성공!

### 3️⃣ 기록 조회 (Top 5)
- **기록 조회 화면**을 통해 오늘 내가 시도했던 사진들 중 **유사도 점수가 가장 높았던 상위 5개의 사진과 점수**를 한눈에 조회할 수 있습니다. 
- 자정이 지나면 오늘의 포즈가 리셋됨과 함께 기록 페이지 안 기록도 리셋됩니다.

<br>

## 🛠️ Tech Stack & Core Logic (핵심 기술)
- **Pose Estimation:** MediaPipe / AI 관절 인식 모델을 통한 고정밀 스켈레톤 좌표 추출
- **Similarity Engine:** 신체 부위별 가중치 기반 벡터 각도 연산을 통한 유사도 판정 엔진 구축
